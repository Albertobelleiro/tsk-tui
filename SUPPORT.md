# Support

## Where to Get Help

- Usage questions: open a GitHub Discussion (recommended)
- Bugs: open a GitHub Issue using the bug report template
- Feature ideas: open a GitHub Issue using the feature request template
- Security concerns: follow [`SECURITY.md`](SECURITY.md)

## Before Opening an Issue

Please include:

- `tsk` version
- Operating system
- Reproduction steps
- Expected behavior vs actual behavior
- Relevant logs/output (with secrets removed)

## Maintainer Response

Response times are best-effort and depend on maintainer availability.

Critical security reports are prioritized.
