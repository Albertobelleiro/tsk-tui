# tsk -- Brand Assets

## Directory Structure

| Folder | Contents |
|--------|----------|
| `logo/svg/` | Vector logos (SVG) -- logomark, logotype, full logo |
| `logo/png/` | Rasterized logos -- `{name}-{size}.png` (e.g., `logo-128.png`) |
| `logo/favicon/` | Favicons -- `favicon.ico`, `favicon-{size}.png`, `apple-touch-icon.png` |
| `icons/` | App icons and terminal icon variants |
| `social/` | Social media images -- `og-image.png` (1200x630), `twitter-card.png` |
| `screenshots/` | Terminal screenshots for docs and website |
| `palette/` | Color palette swatches and Tokyo Night reference |

## Naming Convention

- Lowercase, hyphen-separated: `tsk-logo-dark.svg`
- Include variant in name: `-dark`, `-light`, `-mono`
- Include size for PNGs: `-128`, `-256`, `-512`
